Welcome to the Randomizer Pokedex!

This tool was created to help explore randomized Pokemon games. It can be during play to help keep track of information, or after a playthrough to analyze all of the different pokemon.

I made this because my favorite part of Pokemon randomizers is exploring a brand new world filled with entirely "new" Pokemon. Think of this as a sort of Pokedex you can carry with you on a journey, filling it in as you go. For example, the way I use it is I reveal types immediately upon encountering any Pokemon. After defeating a Pokemon at least once, I reveal its abilities and stats. Everyone is free to come up with their own way to reveal information—I tried to make the tool flexible.

Usage

This site is configured to accept spoiler log files generated by the Universal Pokemon Randomizer ZX. To begin, click the upload button on the home page or on the pokedex page and select the spoiler log generated when you used the randomizer (be sure to generate the spoiler log when randomizing your game).

You can use the search bar in the "Pokemon" tab to select any pokemon to view. By default, the type, stat, ability and other information about each pokemon is hidden. Any unknown information can be revealed by clicking on it. The page will remember what informtion you reveal about which pokemon. You also have the option to reveal all information about every pokemon by clicking the Spoil button at the top.

The site will auto-save any progress you make, but you can also click the save button, which will download a .pkdx file with all of the log's information and what you've revealed so far. Next time you visit the page, it will remember the last log file you were working with, but you can also upload a .pkdx file manually to restore that way.

For extra fun, you can try out my version of the Pokemon Randomizer which enables extra features, like randomized Base Stat Totals, and extra gym challenges!

This page is still under development, so not all information from the log file can be displayed yet. In no particular order, I would like to add the following features:

  • Matchup information
  • A Team Builder/Analysis section

Known Issues

  • Some of the images can have a slight jitter when loading in (especially in Type sections, and cases where a Pokemon has multiple Evolutions)
  • This page will not work with extremely old versions of the Randomizer. I will not fix this. It should be compatible with most recent versions of the Randomizer (4.0.0 and up)
  • Move Tutor and Breeding Moves are not currently displayed. These shouldn't impact a randomized playthrough too much.
